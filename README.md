# BlinkIT Sales & Performance Dashboard

## Project Overview

This Power BI project analyzes BlinkIT retail sales data to uncover insights into sales performance, outlet efficiency, product trends, and customer preferences. The dashboard provides an interactive view of key business metrics and supports data-driven decision-making.

## Problem Statement

Retail businesses generate large volumes of sales data, making it challenging to monitor performance and identify growth opportunities. This project aims to provide a centralized dashboard for tracking KPIs and generating actionable insights.

## Key Features

- Total Sales Analysis
- Average Sales per Outlet
- Average Customer Rating
- Sales by Outlet Type
- Sales by Outlet Location
- Sales by Item Type
- Sales by Item Fat Content
- Product & Outlet Performance Analysis
- Interactive Filters and Slicers

## Tools & Technologies

- Power BI
- Power Query
- DAX
- Data Modeling
- Data Visualization

## Key Insights

- Total Sales: $1.20M
- Average Sales per Outlet: 75.11K
- Average Rating: 3.92
- Fruits & Vegetables emerged as a top-selling category.
- Supermarket Type 1 generated the highest sales contribution.
